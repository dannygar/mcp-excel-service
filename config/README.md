# Configuration Directory

This directory holds environment configuration files for local development.

**Files stored here:**
- `.env.local` - Azure AD credentials for local development (auto-generated by `scripts/register-app.ps1`)

**Note:** All `.env*` files in this directory are excluded from git via `.gitignore`.

## Setup

Run the app registration script to create credentials:

```pwsh
.\scripts\register-app.ps1
```

This will create `config/.env.local` with your Azure AD credentials.
